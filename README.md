## 📊 Velora Superstore – Sales & Profit Performance Analysis

1️⃣ Project Title
Velora Superstore: Executive Sales & Profit Performance Dashboard
An end-to-end data analytics project analyzing sales performance, profitability drivers, discount impact, and regional trends using Python, SQL concepts, and Power BI.

2️⃣ Project Overview
This project analyzes multi-year retail transaction data (2014–2017) to uncover key revenue drivers, profitability risks, and operational inefficiencies.
The goal is not only to evaluate sales performance but to diagnose margin erosion, discount impact, and category-level financial health.

The analysis combines:
-Data cleaning & EDA in Python
-Business-driven questioning
-Interactive Power BI dashboard
-Profitability risk assessment

3️⃣ Tech Stack
The project was built using:
🐍 Python (Pandas, Matplotlib, Seaborn) – Data cleaning & exploratory analysis
📊 Power BI Desktop – Interactive dashboard development
🧠 DAX – Calculated measures (Profit Margin, KPIs, aggregations)
📂 Power Query – Data transformation
🗂 CSV Dataset – Superstore retail data
📓 Jupyter Notebook – Analytical workflow documentation

4️⃣ Data Source
Dataset: Superstore Retail Dataset
The dataset contains:
Order Date (2014–2017)
Sales
Profit
Discount
Category & Sub-Category
Region
Ship Mode
Customer Segment
It represents retail transactional data used to simulate business intelligence reporting and profitability analysis.

5️⃣ Business Problem

-Whether sales are profitable
-Which products are destroying margin
-Whether discounts are sustainable
-Which regions convert sales efficiently into profit
-High revenue does not always mean high profitability.

This project aims to answer:
-Which category drives the most revenue?
-Which sub-category is causing financial loss?
-Does discounting improve or damage profitability?
-Which region converts sales into profit most efficiently?
-How does profit trend seasonally?

6️⃣ Key Business Questions Answered

-What is the overall sales and profit performance?
-Which category has the highest profit margin?
-Which sub-category contributes most to losses?
-How does discount impact profitability?
-Which region has the strongest sales-to-profit conversion?
-What are the monthly seasonal trends in sales and profit?
